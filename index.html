🎌 Origine

Inventé par Toyota dans les années 1950, le Kanban (看板 = « tableau de signalisation » en japonais) est une méthode de gestion visuelle du flux de travail. Repris par les équipes agiles, il est parfaitement adapté aux projets d'école et aux équipes pédagogiques.

📋 Les 4 colonnes

📥
À faire — Backlog
Toutes les tâches identifiées, pas encore démarrées. C'est le réservoir d'actions à planifier. Ajoutez-y tout ce que vous n'avez pas encore commencé.
⚙️
En cours — In Progress
Tâches actuellement traitées. Règle d'or : limitez à 3–4 cartes maximum pour ne pas se disperser. Si tout est "en cours", rien n'avance vraiment.
👁
En révision — Review
Tâches terminées qui attendent une validation, relecture ou retour d'un pair. Utile pour le travail en équipe : l'autre valide avant de clore.
✅
Terminé — Done
Tâches validées et closes. Cette colonne est motivante : voir les accomplissements s'accumuler donne de l'élan à toute l'équipe !

🏫 Usages concrets à l'école

  • Évaluations nationales : une carte par compétence ou groupe d'élèves à suivre, de l'analyse à la remédiation
  • Projet d'école (PAEJ) : une carte par action, avec responsable et deadline
  • Conseil d'école : préparer, traiter, archiver les points à l'ordre du jour
  • Plan numérique : suivi équipements, formations, déploiements ENT
  • Dispositifs élèves : suivi PPRE, PAP, PAI, PPS par cycle
  • Projets partenariaux : sorties, intervenants, dossiers DSDEN

⚡ 3 règles d'or

  • 🎯 1 carte = 1 action précise, réalisable et mesurable
  • ⚠️ Limiter les "En cours" : 3–4 max pour rester focalisé
  • 🔄 Faire avancer les cartes ensemble à chaque réunion (10 min suffisent !)

🖱 Utilisation rapide

  • Déplacer une carte : glisser-déposer d'une colonne à l'autre
  • Modifier une carte : ✏️ au survol, ou clic sur la carte dans la vue Membres
  • Filtrer : cliquer sur une étiquette dans la barre de filtres
  • Nouvelle tâche : bouton ＋ en bas de colonne, ou ＋ Tâche en haut

🔒 Pourquoi des fichiers JSON ?

Vos données ne sont jamais envoyées sur Internet. Elles restent sur votre ordinateur, dans le navigateur (stockage local). Le format JSON est un fichier texte lisible, sauvegardable et partageable — idéal pour une utilisation RGPD en école.

💡 Le navigateur mémorise vos données automatiquement. Le fichier JSON vous permet de faire des sauvegardes manuelles et de partager le tableau avec des collègues.

💾 Sauvegarder (exporter)

1
Cliquer sur 💾 Sauvegarder
Bouton en haut à droite de l'application, dans la barre noire.
2
Le fichier se télécharge automatiquement
Un fichier nommé EduKanban_École_2025-04-10.json apparaît dans votre dossier Téléchargements. Le nom inclut la date automatiquement.
3
Conservez ce fichier précieusement
Placez-le dans un dossier partagé (Teams, ENT, clé USB) pour que l'équipe puisse y accéder. Pensez à sauvegarder régulièrement — par exemple avant chaque réunion.

⌨️ Raccourci clavier : Ctrl + S (ou Cmd + S sur Mac) déclenche la sauvegarde instantanément, sans cliquer.

📂 Charger (importer)

1
Cliquer sur 📂 Charger
Bouton à côté de Sauvegarder dans la barre noire en haut.
2
Sélectionner votre fichier JSON
Une fenêtre s'ouvre pour choisir un fichier. Naviguez vers votre dossier Téléchargements (ou partagé) et sélectionnez le fichier .json sauvegardé précédemment.
3
Le tableau se recharge automatiquement
Tous vos projets, tâches, membres et étiquettes sont restaurés instantanément. Un message de confirmation apparaît en bas de l'écran.

⚠️ Attention : charger un fichier remplace toutes les données actuelles. Si vous souhaitez conserver l'état actuel, commencez par sauvegarder avant de charger un autre fichier.

🤝 Partager avec un collègue

  • Sauvegardez votre fichier JSON
  • Envoyez le fichier par mail, Teams, ENT ou clé USB
  • Votre collègue ouvre EduKanban dans son navigateur et clique sur 📂 Charger
  • Il voit exactement le même tableau que vous
  • Pour mettre à jour, il sauvegarde son fichier et vous le renvoie

💡 Pour une collaboration simultanée en temps réel (plusieurs personnes en même temps), une version avec base de données en ligne est possible — contactez l'ERUN de votre circonscription.

📋 Tableau Kanban

La vue principale. Glissez-déposez les cartes entre colonnes. Cliquez sur ＋ en bas d'une colonne pour ajouter une tâche. Survolez une carte pour voir les boutons ✏️ et 🗑.

📅 Calendrier

Visualisez toutes les échéances sur un calendrier mensuel. Naviguez mois par mois avec les boutons ‹ et ›. Cliquez sur un événement pour modifier la tâche. Les tâches de tous les projets apparaissent ensemble.

📊 Tableau de bord

Vue synthèse automatique : avancement global, répartition par colonne, par priorité, charge par membre, étiquettes utilisées, tâches en retard, et comparaison de tous les projets. Se met à jour en temps réel.

📄 Ordre du jour

Génère automatiquement un ordre du jour structuré pour vos réunions. Sélectionnez le projet et la date, et l'application organise vos tâches par statut. Cliquez sur 🖨 pour imprimer en A4 paysage avec toutes les informations.

👥 Vue par membre

Une colonne par enseignant/personnel avec toutes ses tâches. Filtrez par projet en haut. Identifiez les surcharges et les membres sans tâches assignées. Les tâches non assignées apparaissent dans une colonne dédiée "Non assigné".

🖨 Imprimer un projet

Cliquez sur 🖨 Imprimer dans la barre noire. Le projet actif s'imprime en A4 paysage avec les 4 colonnes, toutes les cartes, responsables et étiquettes. Aussi accessible via Ctrl + P.

⌨️ Raccourcis clavier

Ctrl + SSauvegarder JSON
Ctrl + NNouvelle tâche
Ctrl + PImprimer le projet
ÉchapFermer une fenêtre

Sur Mac, remplacez Ctrl par Cmd (⌘).

🖱 Gestes souris

  • Glisser-déposer une carte → la déplace dans une autre colonne
  • Survoler une carte → affiche les boutons ✏️ modifier et 🗑 supprimer
  • Survoler un onglet projet → affiche le menu renommer / supprimer
  • Cliquer sur un événement calendrier → ouvre la modification de la carte
  • Cliquer sur une carte en vue Membres → ouvre la modification

📱 Conseils d'utilisation

  • Projetez l'application en réunion d'équipe pour animer le suivi en 10 min
  • Sauvegardez le JSON avant chaque réunion comme "instantané" de l'état du projet
  • Utilisez l'Ordre du jour pour préparer et imprimer l'ordre du jour automatiquement
  • La vue Membres permet de visualiser la charge de travail de chaque enseignant
  • Les tâches ⚠️ en retard apparaissent en rouge dans toutes les vues
